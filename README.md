# shanghai_in_W
웹페이지 소개
주제: 꿈속에서 다시 걷는 상하이 "기억을 따라 흐르는 여행 기록 웹페이지" <꿈속에서 다시 걷는 상하이〉는 여행의 ‘결과’를 나열하는 페이지가 아니라, 여행 전의 설렘부터 마지막 밤의 불꽃까지, 시간의 흐름에 따라 기억을 다시 걷는 경험을 목표로 기획된 인터랙티브 웹페이지이다. 이 웹페이지는 단순한 사진 아카이브가 아닌, 감정·공간·시간이 겹쳐진 개인적인 여행 서사를 시각적으로 풀어내는 데 초점을 둔다.
기획구조: 페이지 전체는 HTML 시맨틱 구조를 활용해 구성되었으며,
태그 안에서 Day 0 → Day 3까지의 타임라인을 따라 자연스럽게 흐른다. Hero Section: 여행의 전체 분위기를 압축적으로 전달하는 도입부로, ‘우리의 꿈같던 상하이’라는 타이틀을 중심으로 꿈속에 들어서는 듯한 감각을 유도한다. Day 0 – 여행 전 설렘: 출발 전 옷 쇼핑, 여행 계획, 짐 싸기 등 ‘아직 떠나지 않았지만 이미 시작된 여행’의 순간을 담아 본격적인 여정의 서막을 연다. Day 1 ~ Day 3: 실제 여행 일정에 맞춰 하루 단위로 섹션을 구성하고, 각 장소를 memory-card 형태로 기록하여 하루의 기억이 카드처럼 차곡차곡 쌓이도록 설계하였다. - 디자인 컨셉: 전체 디자인은 ‘몽환적인 기억의 레이어’를 핵심 키워드로 삼았다. 배경: 파스텔 톤의 conic-gradient를 사용해 오로라처럼 일렁이는 배경을 구현하고,이를 position: fixed로 고정하여 스크롤 중에도 꿈속을 걷는 듯한 연속성을 유지한다. 이미지 표현: 정형적인 사각형 대신 유기적인 타원형 이미지를 사용하고, 필름 카메라 느낌의 필터를 적용해, ‘사진’이 아닌 ‘추억’에 가까운 분위기를 연출한다. -마무리 <꿈속에서 다시 걷는 상하이>는“여행을 기록한다”는 행위를 넘어, 기억을 다시 체험하는 디지털 공간을 만드는 것을 목표로 한다.
*웹페이지의 중요성(목적) 여행 정보를 한눈에 정리해 전달하고, 사용자에게 실제 여행을 간접적으로 체험하게 해 준다는 점에 있다. 일정, 관광지, 음식, 이동 경로 등을 웹페이지로 구성하면 복잡한 정보를 직관적으로 보여줄 수 있고, 여행 전 계획 수립에도 큰 도움을 준다. 또한 시각적 요소와 스토리텔링을 활용해 상하이의 문화·역사·도시 이미지를 효과적으로 전달함으로써, 단순한 정보 나열을 넘어 여행에 대한 흥미와 몰입도를 높이는 역할을 한다.
